# Sleepy-Tyres
Drowsiness Detection and Vehicle Safety System using OpenCV, Mediapipe, and Arduino.

### Features:
- Real-time eye closure & head tilt detection using OpenCV + Mediapipe  
- Arduino hardware integration (LEDs, buzzer, motor speed control)  
- Pulse sensor monitoring for physiological fatigue detection  
- Gradual vehicle slowdown and alert system  

### Tech Stack:
Python | OpenCV | Mediapipe | Arduino | L298N Motor Driver
